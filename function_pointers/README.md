# Function Pointers

This project introduces **function pointers** in C: how to declare them, what they represent in memory, and how to use them to write flexible code (callbacks).

## What we are learning (now)

- How to declare a function pointer (example: `void (*f)(char *)`)
- How to pass a function as an argument and call it safely
- How callbacks can change a function’s behavior without changing its code

## What we should be able to explain by the end

- What function pointers are and common ways to use them
- What a function pointer actually stores (an address)
- Where that address points to (executable code in the program’s memory)
