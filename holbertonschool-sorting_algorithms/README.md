# Sorting Algorithms

This project contains implementations of various sorting algorithms in C, following Holberton School requirements.

## Features
- Bubble sort and more
- Betty style compliant
- No use of standard library functions (except for provided print functions)

## Usage
Compile with:
```
gcc -Wall -Wextra -Werror -pedantic 0-bubble_sort.c print_array.c main.c -o bubble
```
