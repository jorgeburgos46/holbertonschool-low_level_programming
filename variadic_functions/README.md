# Variadic Functions

This project introduces the concept of **variadic functions in C**, which are functions that can accept a variable number of arguments.

To implement these functions, the standard library `<stdarg.h>` is used. This library provides macros that allow access to arguments passed after the fixed parameters.

## Learning Objectives

At the end of this project, you should be able to explain:

- What variadic functions are
- How to use `va_start`, `va_arg`, and `va_end`
- Why and when variadic functions are useful
