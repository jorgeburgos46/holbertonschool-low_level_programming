# File I/O

This project is part of the Holberton School low level programming curriculum.

## Description

This project introduces file descriptors and file input/output in C using system calls.

## Learning Objectives

At the end of this project, you should be able to explain:

- How to create, open, close, read and write files
- What file descriptors are
- What the 3 standard file descriptors are and their POSIX names
- How to use the system calls `open`, `read`, `write` and `close`
- How to use the flags `O_RDONLY`, `O_WRONLY` and `O_RDWR`
- What file permissions are
- What a system call is
- The difference between a function and a system call
