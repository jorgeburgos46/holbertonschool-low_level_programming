# Hash Tables

This project covers the implementation of hash tables in C.

## Learning Objectives

- What is a hash function
- What makes a good hash function
- What is a hash table, how do they work and how to use them
- What is a collision and how to handle collisions (chaining)
- Advantages and drawbacks of using hash tables
- Common use cases of hash tables

## Files

| File | Description |
|------|-------------|
| `hash_tables.h` | Header file with structures and prototypes |
| `0-hash_table_create.c` | Creates a hash table |
| `1-djb2.c` | Implements the djb2 hash function |
| `2-key_index.c` | Returns the index for a key using hash % size |
| `3-hash_table_set.c` | Adds or updates an element in the hash table |
| `4-hash_table_get.c` | Retrieves a value associated with a key |
| `5-hash_table_print.c` | Prints the hash table |
| `6-hash_table_delete.c` | Deletes the entire hash table |
