# Singly Linked Lists

This project is part of the Holberton School Low Level Programming curriculum.

## Description

This project introduces singly linked lists in C.

For this task, the goal is to write a function that prints all the elements of a `list_t` list.

## Files

- `lists.h` - Header file containing the definition of `list_t` and function prototypes
- `0-print_list.c` - Function that prints all the elements of a `list_t` list
